# Greenhex 0.1

Libre Office Impress Theme for openSUSE

![screenshot](screenshot/1.png)

## Required

- libreoffice
- adobe-sourcesanspro-fonts
- fontawesome-fonts

### Install in openSUSE

`$ sudo zypper in libreoffice adobe-sourcesanspro-fonts fontawesome-fonts`
    
## Fiture
- animation
- no bitmap (fully editable)
- lightweight
